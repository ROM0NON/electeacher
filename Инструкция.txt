Инструкция по установке "Электронный учитель"

1. Скачиваем файл python6.py.
2. Устанавливаем python 3.13 и библиотеки с помощью командной строки либо терминала(matplotlib, json, socket, os, tkinter, datetime)
3. Устанавливаем pyinstaller.
4. Вводим в терминал либо в командную сторку, pyinstaller -F -m python6.py
5. Код превращается в приложение .exe (предназначен для Windows)

Инструкция по пользованию "Электронный учитель"

1. Используем режим учитель для создания тестов и их сохранения.
1.1 Сохраняем тест как и для учителя (с ответами) так и для ученика (без ответов).
1.2 Отправляем ученику либо через USB-флеш накопитель, либо через сеть, узнав его IP-адрес и подключение его к порту.
1.3 Анализируем ответы ученика с правильными следующим образом: сначало выбираем файл с ответами ученика, потом с правильными ответами.

2. Используем режим ученик для решения теста.
2.1 Отвечаем на вопросы и сохраняем результат.
2.2 Отправляем учителю так же, как в пункте 1.2.

Готово, вы способны полностью пользоваться программой.
